### 트리 자료구조
#### 이진 탐색 트리: 트리 자료구조 중 가장 간단한 형태
 * 부모 노드보다 왼쪽 자식 노드가 작다.
 * 부모 노드보다 오른쪽 자식 노드가 크다.
 * 즉, 왼쪽<부모<오른쪽
